# Diagramas C4 — AgroConecta

## Nivel 1 — Diagrama de Contexto

Muestra el sistema AgroConecta y cómo interactúan con él los usuarios y
sistemas externos.

```mermaid
C4Context
    title Diagrama de Contexto - AgroConecta

    Person(farmer, "Agricultor", "Pequeño productor que gestiona sus fincas, cultivos y cosechas")
    Person(admin, "Administrador / Docente", "Revisa y evalua el funcionamiento de la plataforma")

    System(agroconecta, "AgroConecta", "Plataforma que permite registrar fincas, cultivos y cosechas via API REST")

    System_Ext(swagger, "Swagger UI", "Interfaz de documentacion y pruebas interactivas")
    System_Ext(postman, "Postman", "Cliente para probar manualmente los endpoints")

    Rel(farmer, agroconecta, "Se autentica y administra sus datos", "HTTPS/JSON")
    Rel(admin, swagger, "Explora y prueba la API")
    Rel(swagger, agroconecta, "Consume", "HTTPS/JSON")
    Rel(admin, postman, "Ejecuta colecciones de prueba")
    Rel(postman, agroconecta, "Consume", "HTTPS/JSON")
```

## Nivel 2 — Diagrama de Contenedores

Muestra los contenedores (aplicaciones/servicios desplegables) que componen
AgroConecta y cómo se comunican entre sí.

```mermaid
C4Container
    title Diagrama de Contenedores - AgroConecta

    Person(farmer, "Agricultor", "Usuario de la plataforma")

    System_Boundary(agroconecta, "AgroConecta") {
        Container(api, "Backend API", "Node.js + Express", "Expone la API REST, aplica MVC + Repository Pattern, valida y autentica peticiones")
        Container(swaggerui, "Swagger UI", "swagger-ui-express", "Documentacion interactiva servida en /api-docs")
        ContainerDb(db, "Base de datos", "PostgreSQL 16", "Almacena Farmers, Farms, Crops y HarvestRecords")
    }

    Rel(farmer, api, "Realiza peticiones REST (JSON)", "HTTPS")
    Rel(farmer, swaggerui, "Consulta documentacion / prueba endpoints", "HTTPS")
    Rel(swaggerui, api, "Genera requests de prueba hacia", "HTTPS")
    Rel(api, db, "Lee y escribe datos via Prisma ORM", "SQL/TCP 5432")
```

## Nivel 2 (detalle interno) — Capas del monolito

Vista adicional que muestra las capas internas del contenedor "Backend API",
útil para explicar el patrón MVC + Repository durante la defensa.

```mermaid
flowchart TD
    A[Cliente HTTP / Swagger / Postman] --> B[Rutas Express]
    B --> C[Middlewares: CORS, Helmet, Auth JWT, Validacion Zod]
    C --> D[Controladores]
    D --> E[Servicios de negocio]
    E --> F[Repositorios]
    F --> G[(PostgreSQL via Prisma)]
```
